Html Tags

  1. Italic Tag : The italic tag defines a part of text in an alternate voice or mood.
    The content inside is typically displayed in italic.
  2. Image Tag : The img tag is used to embed an image in an HTML page.
  3. Bold Tag : The tag specifies bold text without any extra importance.
  4. Paragraph Tag : The tag defines a paragraph.
    Browsers automatically add a single blank line before and after each element.
  5. Section Tag : The tag defines a section in a document.
  6. Table Tag : The tag defines an HTML table.
    An HTML table consists of one element and one or more tr, th, and td elements.
  7. Underline Tag : The "u" tag represents some text that is unarticulated and styled differently from normal text, such as misspelled words or proper names in Chinese text.
    The content inside is typically displayed with an underline.
  8. Title Tag : The title tag defines the title of the document.
    The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
  9. Strong Tag : The strong tag is used to define text with strong importance.
    The content inside is typically displayed in bold.
  10. Small Tag : The small tag defines smaller text (like copyright and other side-comments).
