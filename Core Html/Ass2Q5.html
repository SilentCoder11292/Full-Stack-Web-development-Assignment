MEAN Stack
A full-stack development stack that uses MongoDB for the database, Express.js for the server framework, Angular for the frontend framework, and Node.js for the runtime environment.
MERN Stack
A full-stack development stack that uses MongoDB for the database, Express.js for the server framework, React for the frontend library, and Node.js for the runtime environment.
LAMP Stack
A traditional full-stack stack consisting of Linux as the operating system, Apache as the web server, MySQL as the database, and PHP as the backend programming language.
JAMstack
A modern full-stack architecture that relies on JavaScript (client-side), APIs, and Markup to build fast and secure websites and web applications.
PERN Stack
A full-stack development stack that uses PostgreSQL for the database, Express.js for the server framework, React for the frontend library, and Node.js for the runtime environment.
